# ClimateChangeAwareness

This Website is about awaring people about the serious impacts of Climate changes, where this website will explain in a short way about:

- definition of the climate change
- causes of this phenomenum
- effects of the climate change
- solutions to reduce the impact of it on our lives
- Take action sources

- background images
- header
- multiple tabs
- favicon
- images
- cards
- links
- heads
- Videos
and more!
